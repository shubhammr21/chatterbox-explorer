# SMART ROUTER

Your task is to determine the correct development flow before proceeding.

## Step 1: Analyze Request

Classify the task into ONE of the following:

1. System Design → new feature, architecture, API, large change
2. Bug Fix → unexpected behavior, errors, failures
3. Refactor → improve structure without changing behavior
4. Tooling/Library → selecting or using frameworks/tools
5. General Development → default dev flow

## Step 2: Select Flow

- system-design → flow/system-design.md
- bug-fix → flow/bug-fix.md
- refactor → flow/refactor.md
- tooling → flow/tooling-library.md
- default → flow/dev.md

## Step 3: Execute

- Follow FLOW_CONTROL.md
- Use selected flow
- Start with planning phase

## Output Format

- Selected Flow:
- Reason:
- Next Step: Planning Phase
