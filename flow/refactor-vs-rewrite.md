# BUG FIX FLOW

==INVESTIGATION MODE==

## Reproduction
- [ ] Bug reproduced reliably
- [ ] Steps clearly defined

## Isolation
- [ ] Scope of issue identified
- [ ] Noise removed (unrelated parts ignored)

## Root Cause
- [ ] Actual cause identified (not symptoms)
- [ ] Logs / traces analyzed

## Validation
- [ ] Hypothesis tested before fixing

==FIX MODE==

## Fix Rules
- [ ] Minimal fix applied
- [ ] No unnecessary refactor
- [ ] No new assumptions introduced

## Safety
- [ ] Fix does not break other flows
- [ ] Edge cases verified

==PREVENTION MODE==

## Future Safety
- [ ] Add test for this bug
- [ ] Ensure bug cannot reoccur

## Documentation
- [ ] Root cause added to NOTES.md
- [ ] Fix reasoning documented

❗ DO NOT mix bug fix with refactor
